Guia de las sentencias básicas de FPDF

require('fpdf.php')

Esta línea se utiliza para incluir la librería FPDF en tu script PHP. Debes asegurarte de que la ruta al archivo 'fpdf.php' sea correcta.

$pdf = new FPDF()

Esta función crea una nueva instancia de la clase FPDF, que representa el documento PDF que estás creando.

$pdf->AddPage()

Esta función agrega una nueva página al documento PDF. A partir de este punto, puedes comenzar a agregar contenido a la página actual.

$pdf->SetFont('fuente', 'estilo', tamaño)

Esta función define la fuente, estilo y tamaño de fuente para el texto que se agregará al PDF.

$pdf->SetTextColor(rojo, verde, azul)

Esta función establece el color del texto. Los valores de rojo, verde y azul son números enteros que van de 0 a 255.

$pdf->Cell(ancho, alto, texto, borde, salto_de_linea, alineación)

Esta función agrega una celda al PDF con el texto especificado. Puedes controlar el ancho, alto, borde, si se salta de línea y la alineación del texto en la celda.

$pdf->Image('nombre_de_archivo', x, y, ancho, alto, formato)

Esta función Inserta una imagen en el PDF desde un archivo. Debes especificar el nombre del archivo de la imagen, las coordenadas x e y de la posición de la imagen, el ancho y el alto, y el formato de la imagen (JPEG, PNG, etc.).

$pdf->Output('nombre_del_archivo.pdf', 'I')

Esta función se utiliza para mostrar el PDF en el navegador. Puedes proporcionar un nombre para el archivo resultante y especificar 'I' para mostrarlo en el navegador (Puedes reemplazar 'I' con 'F' para que se descargue el documento de una sola vez).
